
<h1 align="center">METADATA</h1>

<h2>1. 🏦 KedolikSwap LP Token Metadata Repository</h2>

This repository contains the official metadata for **KedoX LP tokens** issued by KedolikSwap.

KedoX LP tokens represent liquidity provider positions in KedolikSwap liquidity pools and are used by wallets, explorers, analytics platforms, and ecosystem integrations to display accurate pool information.

<h2>2. 🌊 Supported LP Tokens</h2>

Current KedoX LP tokens include:

- 🟣 KedoX LP WSOL-KEDOL
- 🔵 KedoX LP USDC-KEDOL
- 🟢 KedoX LP USDT-KEDOL
- 🟠 KedoX LP SOL-USDC
- 🟡 KedoX LP SOL-USDT
- 🔷 KedoX LP USDC-USDT

Additional LP token metadata may be added as new liquidity pools are launched on KedolikSwap.

<h2>3. 📋 Metadata Information</h2>

Each LP token metadata file may contain:

- 🏷️ Token Name
- 🔤 Symbol
- 🔑 Mint Address
- 🔢 Decimals
- 📝 Description
- 🖼️ Logo URI
- 🌊 Pool Information
- ⚙️ Protocol Information
